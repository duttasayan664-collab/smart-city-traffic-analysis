# Smart City Traffic Pattern Analysis

## Project Overview

This project analyzes urban traffic patterns using a Smart City traffic dataset.
The goal is to identify peak traffic periods, analyze traffic distribution, and understand traffic patterns across different time periods.

## Tools Used

* Power BI
* Data Visualization
* Data Analysis

## Dashboard Features

* Total Vehicles KPI
* Average Traffic Analysis
* Monthly Traffic Trends
* Day-wise Traffic Distribution
* Traffic Category (Peak vs Normal)
* Interactive Filters (Month, Day, Junction)

## Key Insights

* Peak traffic occurs during mid-week days.
* June shows the highest monthly traffic volume.
* Normal traffic accounts for the majority of traffic observations.
* Traffic decreases significantly during weekends.
